🌸 Clop Garden

A tactile, auditory, and highly satisfying digital fidget toy.
Clop Garden is a lightweight web application designed for the simple joy of creating, dropping, and zapping colorful "jelly blocks." It features custom-synthesized sound effects, smooth physics, and a hidden "Shloop" portal for a minimalist cleanup experience.

🕹️ How to Play

CLOP: Tap to spawn a block in the air.
Turbo Mode: Press and hold the CLOP button to unleash a rapid-fire rain of blocks.
PLOP: Watch as every floating block falls to the garden floor with a satisfying, squishy impact.
SHLOOP: Once you've created a mess, the SHLOOP portal will appear at the bottom. Tap it to vacuum all blocks out of existence with a high-velocity "zip" sound.

✨ Features

Satisfying Physics: Blocks feature "jelly-like" squash and stretch on impact and spawn.
Adaptive UI: The "Shloop" button only appears when there is something to clean up, keeping your view pristine.
Synthesized Audio: No MP3 files required. All sounds (Clops, Plops, and Shloops) are generated in real-time using the Web Audio API.
Mobile Optimized: Built with touch-events and "no-menu" overrides to ensure Turbo mode works perfectly on smartphones.
Neumorphic & Glassmorphism Design: A blend of soft UI and frosted glass aesthetics for a modern, organic look.

🛠️ Installation & Deployment

Since the entire application is contained within a single index.html file, deployment is instant.

Local: Save the code as index.html and open it in any modern browser.
Web:
Drag the folder containing the file into Netlify Drop.
Or upload to GitHub Pages for permanent hosting.

📂 Technical Stack

Language: HTML5, CSS3, Vanilla JavaScript.
Sound: Web Audio API (Oscillators: Sine, Triangle, Sawtooth).
Typography: Quicksand via Google Fonts.
